What is Compression?

Compression reduces the size of files like html, css, javascript, mp3, mp4, jpg before theyre sent over the internet. Smaller files load faster which helps improve website performance and saves on bandwidth. there are different ways to compress files but the most common ways are online compressor sites and free to download compression software

  • Lossy Compression: Reduces file size by removing some data which can effect the quality.
  • Lossless Compression: Reduces the file size without loasing any data during the process. e.g. ZIP/RAR files
